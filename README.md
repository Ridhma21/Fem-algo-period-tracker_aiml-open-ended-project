# FEM-ALGO Wellness App

## Description

FEM-ALGO is a Python-based GUI application designed as a simple wellness companion. It helps users track menstrual cycles, analyze mood, check basic health symptoms, and access useful features like graphs and doctor suggestions.

The application is built using Tkinter and focuses on providing an interactive and easy-to-use interface.

## Features

* User profile setup (name, age, height, weight)
* Menstrual cycle tracking with average cycle calculation
* Graph visualization of cycle trends using matplotlib
* Basic PCOS/PCOD risk checker based on symptoms
* Mood analysis with suggestions
* Quick access to music platforms for relaxation
* Simple in-app shop for essentials
* Doctor suggestions with booking option
* Clean GUI using Tkinter

## Technologies Used

* Python
* Tkinter (GUI)
* Matplotlib (graphs)
* Webbrowser module

## How to Run

1. Make sure Python is installed

2. Install required libraries:
   pip install matplotlib

3. Run the file:
   python your_file_name.py

4. Enter your details and use the sidebar to navigate through features

## Output

* Displays an interactive GUI application

## Logic Used

* Basic data storage using Python classes
* Average cycle calculation
